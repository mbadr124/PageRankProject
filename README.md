# CSE455-High-Performance-Computing
Distributed PageRank algorithm implementation using shared memory and message passing paradigms

This project contains 3 different variations of the PageRank algorithm, each code file can run using a predefined graph or a randomly generated one (for which you will need to adjust the values of the global variables of# of nodes). The goal of this project is to explore the effects of parallel computing in enhancing the performance of popular algorithms, while gaining a hands-on experience of writing such code. The three code files are nearly identical except for the PageRank function implementation in each one.

The .gexf file could be used with applications like gephi to visualize what a 2000-node graph (which the algorithm works on) looks like.

The OpenMP and MPI codes require multiple steps to run which you could look for online as they depend on your OS.
